## 꽃가루 수치 알리미 어플

이 어플은 꽃가루에 예민한 비염인들에게 도움을 주기 위해 제작하였습니다.

- ### 제공 꽃가루 정보
  
  - 참나무 (4 ~ 6월 제공)
  - 소나무 (4 ~ 6월 제공)
  - 잡초류 (8 ~ 10월 제공)
